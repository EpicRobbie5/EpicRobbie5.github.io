# Security Policy

## Supported Versions

This project is a personal portfolio website. Security updates are provided for the current version.

## Reporting a Vulnerability

If you discover a security vulnerability in this repository, please report it to:

**Email:** robbe.schoenmaekers@student.rhizo.be

Please do not open public issues for security vulnerabilities. This allows us to fix the issue and release a security update before discussing it publicly.

## Security Considerations

- This is a static HTML/CSS website hosted on GitHub Pages
- No backend or database is used
- Dependencies are minimal
- All code is reviewed before deployment

## Contact

For security concerns, please contact the repository owner directly.
